# 🌐 Personal Portfolio Website (Full Stack)

## 📌 Project Description

This is a full-stack personal portfolio website developed to showcase my skills, projects, and contact information. The project includes both frontend and backend functionalities, providing an interactive and dynamic user experience.

The website allows users to explore my projects, view my skills, and contact me through a form that is handled by the backend server.

---

## 🚀 Features

* 🏠 Home page with introduction
* 👨‍💻 Projects showcase section
* 🧠 Skills display
* 📊 Dashboard (if applicable)
* 🔐 Login system
* 📩 Contact form with backend integration
* 📱 Responsive design
