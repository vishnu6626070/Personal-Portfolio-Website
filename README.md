# 🌐 Full Stack Personal Portfolio Website

## 📌 Description

This is a full-stack personal portfolio website designed to showcase my skills, projects, and personal details. It includes both frontend and backend functionalities, allowing users to interact through features like contact forms and login systems.

---

## 🚀 Features

* Personal introduction
* Projects showcase
* Skills section
* Contact form with backend
* Login and dashboard pages
* Responsive design

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js

---

## 📁 Project Structure

PORTFOLIO-PROJECT
│── backend
│── frontend

---

## ⚙️ Setup Instructions

### Clone Repository

git clone https://github.com/your-username/portfolio-project.git

### Run Backend

cd backend
npm install
node server.js

### Run Frontend

Open index.html in browser
